Naviguider-SimpleSerial
=======================

**Naviguider-SimpleSerial** is a library for encoding commands and decoding 
responses using the PNI Sensor SimpleSerial ASCII protocol used by their 
Naviguider-UART and Headway products. It is a simple library for wrapping 
commands with more memorable function names and converting responses into 
dataclasses for easier handling.
